# Tooltip

A brief, informative message that appears when a user hovers over an element.

## Usage

```tsx
import { Tooltip } from 'glasswave';

function App() {
  return (
    <div className="p-4">
      <Tooltip content="This is a tooltip">
        <button>Hover me</button>
      </Tooltip>
    </div>
  );
}
```
