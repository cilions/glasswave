## Usage

```tsx
import {
  InputGroup,
  InputGroupAddon,
  InputGroupInput,
  InputGroupText,
} from "glasswave";

<InputGroup>
  <InputGroupAddon align="inline-start">
    <InputGroupText>https://</InputGroupText>
  </InputGroupAddon>
  <InputGroupInput className="ps-[4.5rem]" />
</InputGroup>
```

## Parts

| Part | Notes |
|------|--------|
| `InputGroup` | Relative flex wrapper |
| `InputGroupAddon` | `align`: `"inline-start"` \| `"inline-end"`; `pointer-events-none` for text addons |
| `InputGroupInput` | Glass input; pad inline start/end to clear addons |
| `InputGroupText` | Muted label inside addon |
