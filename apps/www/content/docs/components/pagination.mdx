## Usage

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from "glasswave";
```

## Props

`PaginationLink` extends `<a>` with `isActive` to highlight the current page (uses primary button variant).

`PaginationPrevious` / `PaginationNext` render full anchor elements inside `PaginationItem`; pass `href` or `onClick` as needed.
