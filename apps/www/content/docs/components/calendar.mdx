## Usage

```tsx
import { Calendar } from "glasswave";
import { useState } from "react";

const [date, setDate] = useState<Date>();

<Calendar mode="single" selected={date} onSelect={setDate} />
```

The component imports `react-day-picker/style.css` internally. All [DayPicker props](https://daypicker.dev) are supported (`mode`, `selected`, `disabled`, `modifiers`, etc.).

## Styling

Glass styling is applied via `classNames` merged with `getDefaultClassNames()`. Override any key through the `classNames` prop.
