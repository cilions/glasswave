# GlassWave

## Installation

```sh
npm install glasswave
# or
yarn add glasswave
# or
pnpm add glasswave
# or
bun add glasswave
```

## Usage

### Button Component

```jsx
import { Button } from 'glasswave';

function App() {
  return (
    <div className="p-4">
      <Button>Default Button</Button>
    </div>
  );
}
```
