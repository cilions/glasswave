---
title: Radio Group
description: Radio group button for single selection.
---

## Usage

```tsx
import { RadioGroup, RadioGroupItem } from "glasswave";
```

```tsx
<RadioGroup defaultValue="a">
  <div className="flex gap-4">
    <RadioGroupItem value="a" />
    <RadioGroupItem value="b" />
  </div>
</RadioGroup>
```
