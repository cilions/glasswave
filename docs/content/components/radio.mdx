# Radio

A control that allows the user to select one option from a set of mutually exclusive options.

## Usage

```tsx
import { Radio } from 'glasswave';

function App() {
  const [selected, setSelected] = useState('value1');

  return (
    <div className="p-4">
      <Radio name="option" value="value1" checked={selected === 'value1'} onChange={setSelected} />
    </div>
  );
}
```
